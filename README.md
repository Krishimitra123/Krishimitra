# KrishiMitra v3.0

KrishiMitra is a deep-tech, RAG-grounded Voice-AI agricultural assistant designed for Karnataka farmers. It bridges ancient science (Vruksha Ayurveda) and modern verified practices (Subhash Palekar ZBNF) to provide hallucination-free, voice-first guidance on soil fertility and crop management.
